Bem-vindo ao Projeto EDOO - Dijkstra

Implementação do algoritmo de Dijkstra para encontrar o menor caminho em grafos, utilizando C++ e programação orientada a objetos (OOP).

Objetivo do Projeto

Este projeto tem como objetivo implementar o algoritmo de Dijkstra em C++, utilizando os conceitos de Programação Orientada a Objetos (POO).

O foco é criar um código bem estruturado, modular e reutilizável.

O algoritmo de Dijkstra é amplamente usado para encontrar o caminho mais curto entre um nó de origem e todos os outros nós em um grafo ponderado, onde os pesos das arestas são não negativos.

Este tipo de problema é crucial em várias áreas da computação e engenharia, e o projeto busca fornecer uma solução otimizada e prática para esse desafio.

Resumo do Algoritmo

O algoritmo de Dijkstra implementado neste projeto encontra o menor caminho entre dois nós em um grafo ponderado, utilizando uma estrutura eficiente de filas de prioridade.
