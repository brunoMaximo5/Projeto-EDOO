Bem-vindo ao Projeto EDOO - Dijkstra

Implementação do algoritmo de Dijkstra para encontrar o menor caminho em grafos, utilizando C++ e Programação Orientada a Objetos (POO).

Objetivo do Projeto

Este projeto tem como objetivo implementar o algoritmo de Dijkstra em C++, utilizando os conceitos de Programação Orientada a Objetos (POO).

O foco é criar um código bem estruturado, modular e reutilizável. O algoritmo de Dijkstra é amplamente usado para encontrar o caminho mais curto entre um nó de origem e todos os outros nós em um grafo ponderado, onde os pesos das arestas são não negativos.

Este tipo de problema é crucial em várias áreas da computação e engenharia, e o projeto busca fornecer uma solução otimizada e prática para esse desafio.

Demonstração do Código:

Na imagem acima, vemos a estrutura do grafo utilizado, onde cada nó possui conexões com outros nós e os respectivos pesos das arestas.

O usuário insere o nó de origem e o nó de destino, e o algoritmo de Dijkstra calcula o menor caminho.

Exemplo de execução:

  • Origem: 50
  • Destino: 10
  • Menor caminho encontrado: 50 → 10
  • Custo total: 9

O sistema exibe os detalhes da busca e permite novas consultas até que o usuário decida sair.

Resumo do Algoritmo

O algoritmo de Dijkstra implementado neste projeto encontra o menor caminho entre dois nós em um grafo ponderado, utilizando uma estrutura eficiente de filas de prioridade.

O usuário pode modificar o grafo conforme desejar, alterando as conexões e pesos diretamente no arquivo grafo.txt.

Também é possível utilizar diferentes grafos, bastando substituir o arquivo ou editar os dados nele contidos.
